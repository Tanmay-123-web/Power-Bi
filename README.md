# 📊 Power BI Dashboard Projects

This repository contains multiple interactive **Power BI dashboards** created for data analysis, visualization, and business insights.  
These dashboards demonstrate skills in **data modeling, DAX, visualization design, and storytelling**.

---

## 🚀 Dashboards Included

### 1️⃣ AW Sales Dashboard
**Description:**
- Sales performance analysis with dynamic date filters
- Monthly revenue, orders, and returns tracking
- Category & subcategory-wise order distribution
- Top products by orders and profits
- Interactive region-wise map (Europe, North America, Pacific)

**Key Skills Used:**
- DAX measures
- Time intelligence
- Drill-through & navigation buttons
- KPI cards & custom visuals

---

### 2️⃣ Corporate Spend Analysis Dashboard
**Description:**
- Year-to-date IT spend analysis
- Actual vs Plan comparison by IT Area
- Variance % analysis by country/region (map visualization)
- Period-wise spend trend analysis
- Sankey visual for variance flow analysis

**Key Skills Used:**
- Advanced DAX calculations
- Financial variance analysis
- Custom themes & dark UI design
- Business-focused data storytelling

---

### 3️⃣ Netflix Content Analysis Dashboard
**Description:**
- Overview of Movies & TV Shows available on Netflix
- Total titles, movies, shows, adult content, and IMDB score KPIs
- Country-wise content distribution
- Genre-wise and age certification analysis
- Release year trend analysis
- Interactive navigation (Movie / Show toggle)

**Key Skills Used:**
- Data cleaning & transformation
- Interactive slicers & navigation
- Visual hierarchy & dashboard UX
- Trend and categorical analysis
